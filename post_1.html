Setting Up an EC2 Instance

Posted by Andrew Huebert on Feb 14, 2020

Setting up an EC2 Instance was actually much easier than I anticipated. To begin with you need to creat an AWS account that will house all of your AWS services. These services include, but are not limited to, EC2, S3 and Lightsail.

Once an AWS account is created you are granted what's called the "Free Tier" of services. Within this "Free Tier" the EC2 service is available. You simply click on EC2, and it will take you to a new Dashboard for all of your EC2 needs.

Once in this Dashboard there is an option that says "Create Instance." You press that button and it will redirect you to a Wizard, similar to a Windows set-up wizard, that will direct you on the options for creating your EC2 instance.

The main thing to focus on is the "AMI" that is chosen. This AMI is the operating system that gets booted onto the EC2 server. Amazon offers their propietary AMI, which is a perfectly fine OS to test this EC2 instance with.

Once you finish the Set-Up Wizard, choosing storage requirements and setting tags, you have completed the first set up of an EC2 instance. This instsance, depending on which protocols and services you included, is now SSH ready, able to host a Web service or even able to be setup as a database.
