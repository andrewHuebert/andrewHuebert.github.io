Connecting a PS4 Controlled to PC

Posted by Andrew Huebert on Apr 24, 2020

Setting up a controller connected to the PC is entirely not too difficult. The PS4 controller has Bluetooth capabilities that make it easy to connect to the computer as long as Bluetooth is enabled.

Connecting the controller to the Epic Games platform, however is much more difficult of a process. It requires extra programming to be mapped correctly to all the controls.

The Epic Games platform does not natively accept controllers, simply the keyboard. However with the DS4Windows program you are able to set up a controller with the platform.

The only issue that I've run into with connecting these two is that Epic Games recognizes the PS4 controller as an XBOX controller. While this isn't a huge issue some of the buttons had to be remapped in order match the PS4 controls and be used properly. This was easy within the DS4 interface. Once all the buttons are mapped properly the controller worked flawlessly.
